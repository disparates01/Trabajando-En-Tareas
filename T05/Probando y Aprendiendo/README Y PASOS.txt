1. Crear la interfax con QtDesigner

2. Guardar el archivo '.ui' 

3. Abrir el prompt de windows en ese directrio. (Se puede apretar 'alt + d' y luego escribir cmd en el mismo directorio)

4. Escribir en el prompt: <nombre archivo.ui> -o <nombre archivo python>. Esto creará el archivo .py con las caracteristicasde la GUI.

5. Crear otro archivo '.py' que contenga los metodos necesarios para iniciar la aplicacion.
